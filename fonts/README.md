# Font Material Symbols - Locale

Questa cartella contiene i font Material Symbols scaricati localmente per garantire che l'applicazione funzioni anche senza connessione internet.

## File inclusi:

- **MaterialSymbolsOutlined.woff2**: Font principale Material Symbols Outlined
- **material-symbols.css**: Definizioni CSS per l'uso locale del font

## Icone utilizzate nell'applicazione:

- `ads_click` - Strumento selezione
- `draw` - Strumento spline  
- `square` - Strumento rettangolo
- `circle` - Strumento cerchio
- `save` - Salva progetto
- `folder_open` - Carica progetto
- `settings` - Impostazioni
- `dark_mode` / `light_mode` - Toggle modalità scura/chiara

## Vantaggi della soluzione locale:

✅ Funziona offline  
✅ Caricamento più veloce  
✅ Nessuna dipendenza esterna  
✅ Privacy garantita (nessuna chiamata a Google Fonts)  
✅ Controllo completo sui font utilizzati
